# Научиться учиться

Данный проект представляет собой одностраничный лендинг, с использованием, изученных в **Яндекс Практикуме**,  технологий **CSS** и **HTML**.

## Используемые технологии 

- БЭМ нейминг
- Файловая структура БЭМ Nested
- Flexbox
- Анимации
- Псевдоэлементы
- Интерактивные элементы
- и тд
